The Chrome extension turns every café you scroll past into a learning
signal for Maroa.

## What it does

- Right-click any IG / FB / TikTok post → **Save to Maroa as inspiration**
- Right-click any selected text → **Save selected text as a Maroa claim**
- Saved entries land in your `marketing_graph_entities` table as
  `type='inspiration'` or `type='claim'`
- The cold-start corpus picks them up on the next pretrainer sweep
- Next week's drafts will reflect the patterns you saved

## Install

Currently developer-only. Until the Chrome Web Store listing is live:

1. Clone the [maroa-api repo](https://github.com/idealbekteshi06/maroa-api)
2. Open Chrome → `chrome://extensions`
3. Toggle **Developer mode**
4. Click **Load unpacked** → pick `browser-extension/`
5. Click the Maroa icon in the toolbar → paste your API URL + token

When the Chrome Web Store listing is live (target: end of May 2026), it's
a one-click install.

## Permissions

- `contextMenus` — to register the right-click items
- `activeTab` — to read the URL of the page you right-clicked on
- `storage` — to remember your API URL + token
- `notifications` — to confirm a save happened

Hosts allowed: Instagram, Facebook, TikTok, Threads, and the Maroa API.
No `<all_urls>`. No tracking. No telemetry.

## Privacy

The extension only talks to the API URL you configure. Saved entries are
visible to your Maroa workspace owner only — RLS enforced server-side.

## Endpoint it calls

```
POST /api/inspiration/save
Authorization: Bearer <token>
Idempotency-Key: <auto-generated per save>

{
  "source": "browser_extension",
  "source_url": "https://instagram.com/p/...",
  "image_url": "https://...",
  "excerpt": "...",
  "captured_at": "2026-05-20T15:30:00Z"
}
```

Returns `{ ok: true, id: '...' }`.
